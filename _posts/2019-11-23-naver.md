# 웹크롤링 -네이버

웹크롤링은 어렵지만 제일 재밌는 것 같다.

```ruby
import requests
from bs4 import BeautifulSoup

response=requests.get('https://www.naver.com/')
html=response.text
soup=BeautifulSoup(html, 'html.parser')
ranking=1

for a in soup.select('span[class=ah_k]'):
    if ranking == 21:
        break
    print(str(ranking) + '위 : ' + a.text)
    ranking = ranking + 1
```
